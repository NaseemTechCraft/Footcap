# Footcap

## Overview

Footcap is a responsive, single-page front-end website designed to showcase footwear products and brand information. It provides a clean landing experience with product visuals, feature highlights, and a contact section to engage visitors.

## Features

- Responsive layout for mobile and desktop
- Product/gallery showcase using optimized images
- Smooth navigation and section scrolling
- Contact/info section for customer inquiries
- Easy customization of styles and content

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Installation

1. Clone the repository.
2. Navigate to the project directory.
3. Install required dependencies (none required for this static site).
4. Open `Index.html` in a browser or use a live server extension.

## Usage

Open the site and use the navigation to explore sections: product gallery, features, and contact. Replace images in `Assets/Images/` and edit styles in `Assets/CSS/Style.css` to customize appearance.

## Screenshots
